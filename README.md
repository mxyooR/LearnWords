# 背单词助手

一个桌面端英语单词学习工具，基于 PyQt5 开发。

## 功能说明

### 1. 单词导入

支持三种导入方式：

**方式一：带含义导入**
```
单词,含义
apple,苹果
banana,香蕉
```

**方式二：自动查词导入**
- 直接输入单词列表（空格、逗号或换行分隔）
- 程序自动联网查询释义和例句
- 支持从文件或剪贴板导入

**方式三：从文件导入**
- 支持 .txt 和 .csv 格式
- 自动识别单词并去重

### 2. 复习系统

**每日任务构成：**
- 所有未复习的单词（复习次数 = 0）
- 所有复习中的单词（复习次数 1-2 次）
- 已掌握的单词（复习次数 ≥ 3 次）按艾宾浩斯遗忘曲线：
  - 第3次复习：上次复习后 2 天
  - 第4次复习：上次复习后 4 天
  - 第5次复习：上次复习后 7 天
  - 第6次复习：上次复习后 15 天
  - 第7次及以后：上次复习后 30 天
  - 每天至少 10 个已掌握单词（不足则随机补充）
  - 每天最多 60 个单词（优先保证未复习和复习中的）

**复习流程：**
1. 点击"开始复习"
2. 看到单词后输入中文含义
3. 点击"检查答案"或"显示答案"
4. 查看释义和例句
5. 选择"我会了"或"我还不会"
   - 我会了：复习次数 +1，更新复习时间
   - 我还不会：只更新复习时间，不增加次数

### 3. 例句功能

- 导入单词时自动获取英文例句
- 例句来源：有道词典、Free Dictionary API、Tatoeba 例句库
- 复习时可点击"显示例句"查看
- 支持滚动查看多个例句

### 4. 数据管理

**统计页面功能：**
- 按状态筛选：全部/未复习/复习中/已掌握
- 查看单词详情：释义、例句、复习次数、上次复习时间
- 双击单元格编辑释义和例句
- 点击"保存修改"保存更改
- 批量删除或清空单词

**数据导出：**
- 导出为 CSV 格式
- 包含单词和详细含义

## 安装使用

### 直接运行源码

```bash
# 安装依赖
pip install PyQt5

# 运行
python word_app.py
```

### 打包为 exe（可选）

```bash
# 安装 Nuitka
pip install nuitka

# 打包（需要准备 icon.ico 图标文件）
python -m nuitka --standalone --onefile --windows-console-mode=disable --enable-plugin=pyqt5 --windows-icon-from-ico=icon.ico --output-dir=output word_app.py
```

## 数据存储

所有数据保存在 `words_data.json` 文件中，包含：
- 单词
- 释义
- 例句列表
- 复习次数
- 最后复习时间

## 系统要求

- Python 3.7+
- PyQt5
- 网络连接（用于自动查词）
